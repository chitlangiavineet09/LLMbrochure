# LLMbrochure

LLMbrochure is a lightweight AI tool that turns a company website into a draft brochure. Given a website URL and a target language, it collects key content from the site, uses an LLM to identify the most relevant supporting pages, and generates a structured brochure in markdown.

## What This Product Does

- reads a company website and extracts meaningful business content
- identifies high-value pages such as About, Company, or Careers
- generates brochure-style copy for clients, investors, candidates, or internal stakeholders
- supports multiple output languages through a simple interactive interface

## Ideal Use Cases

- quick company overviews for sales or partnership preparation
- first-draft brochures for research, recruiting, or investor communication
- rapid content generation from publicly available website information

## Current Scope

This project is best understood as a practical prototype: it automates website scraping, page selection, and brochure generation in a single workflow, making it useful for experimentation, internal tooling, and fast first drafts that can be reviewed and refined.
